# Ego Handmade Human Operator Trial v1 - SPEC

## Goal

Create a candidate-local human operator trial for `Ego_handmade`. The goal is
to capture whether a real user session feels better in natural understanding,
memory behavior, tool gating, trace readability, and failure recovery.

## Authority Snapshot

- formal EGO mainline remains `subject_system_v1_governed_proactivity`
- `Ego_handmade` remains replacement candidate / operator cut only
- `EgoCore`, `OpenEmotion`, and `ego_desktop_lab` remain reference/fallback
- no program-state or evidence-ledger update is authorized

## Contract

- trial scenarios cover 15-20 real operator prompts
- observations are imported as JSONL and reported as JSON + Markdown
- no observation can upgrade `Ego_handmade/memory` into canonical EGO memory
- NoLLM / fake-provider runs are smoke only and cannot prove natural understanding
- a pass requires enough human observations, low correction rate, no memory misuse,
  and no gate violation

## Claim Ceiling

`Ego_handmade human operator trial local candidate report`.

This task cannot claim EGO mainline replacement, formal long-term memory
efficacy, live autonomy, runtime efficacy, stable user benefit, or consciousness.
