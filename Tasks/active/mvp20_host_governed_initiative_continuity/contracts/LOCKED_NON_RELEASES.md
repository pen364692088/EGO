# WP15 / MVP20 Locked Non-Releases

- live autonomy
- OpenEmotion direct reply authority
- tool execution authority
- broader transport claims
- transport enable-policy takeover
- direct proactive send authority
- autonomous outreach
- direct response-plan injection
- direct mutation of `WP7~WP14` owner state
- authority escalation bypass

即使后续 `WP15` 拿到 controlled `E5`，这些 non-releases 也不会自动解锁；必须依赖新的 authority package。
