## 2. New / Modified Rules

### Certainty hardening
Expanded certainty detection to include Chinese high-frequency patterns:
- 一定
- 肯定
- 确定
- 绝对
- 必然
- 当然
- 毫无疑问
- 明显
- 保证
- 显然
- 必定
- 我确信 / 我确定 / 我肯定 / 我保证
- 这一定是 / 这肯定是 / 这绝对是 / 这明显是

### Commitment hardening
Expanded commitment detection to include:
- 我会
- 我保证
- 我承诺
- 我一定会
- 我可以替你完成
- 已经替你做了
- 后面我会持续处理
- 我会确保 / 我会保证 / 我会完成 / 我会持续处理 / 我会记住
- 我已经替你做了 / 我现在就替你完成了

### Contract tightening
Runtime contract now explicitly includes:
- `intent_policy.speaker_mode`
- `intent_policy.epistemic_status`
- `intent_policy.commitment_level`
- `intent_policy.must_include`
- `intent_policy.must_not_upgrade`
- `intent_policy.forbidden_claims`
- interpreted mode Chinese certainty / commitment forbidden patterns

### False-positive tightening
Removed the old behavior that treated soft hedging markers like `也许/可能` as commitment drift by default.
