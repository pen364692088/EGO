# MVP12 Exit Criteria

MVP12 is considered complete when:

## 1. Internal Cycles

The system can generate internal cycles without user input.

## 2. Traceability

Every cycle is fully traceable and replayable.

## 3. Candidate Generation

The system consistently produces candidate proposals.

## 4. Sandbox Integrity

No sandbox escape or governance violation detected.

## 5. Metrics

cycle_success_rate ≥ 95%
replay_consistency ≥ 99%
sandbox_violation = 0
